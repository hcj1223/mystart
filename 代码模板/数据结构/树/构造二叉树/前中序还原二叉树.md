# 前中序还原二叉树

```c++
tree createtree(char *pre, char *mid, int n)
{
    if (n <= 0)
        return NULL;
    tree t = new treenode;
    t->data = *pre;
    int rootpos = -1;
    for (int i = 0; i < n; i++)
        if (mid[i] == *pre)
        {
            rootpos = i;
            break;
        }
    t->lchild = createtree(pre + 1, mid, rootpos);
    t->rchild = createtree(pre + rootpos + 1, mid + rootpos + 1, n - rootpos - 1);
    return t;
}
```
