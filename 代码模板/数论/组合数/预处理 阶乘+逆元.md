# 预处理 阶乘+逆元

```c++
int e[N];
int ie[N]; // 逆元
void init()
{
    e[0] = ie[0] = 1;
    for (int i = 1; i < N; i++)
    {
        e[i] = (ll)e[i - 1] * i % mod;
        ie[i] = (ll)ie[i - 1] * qmi(i, mod - 2, mod) % mod;
    }
}
ll C(int a, int b)
{
    return (ll)e[a] * ie[b] % mod * ie[a - b] % mod;
}
```
