# 判断质数

```c++
bool is_prime(int n)
{
    if (n <= 1)
        return 0;
    for (int i = 2; i <= n / i; i++)
        if (n % i == 0)
            return 0;
    return 1;
}
```

```c++
// x <= 1e18
bool is_prime(long long x)
{
    if (x <= 3)
        return x > 1;
    if (x % 6 != 1 && x % 6 != 5)
        return 0;
    for (int i = 5; i <= x / i; i += 6)
        if (x % i == 0 || x % (i + 2) == 0)
            return 0;
    return 1;
}
```
