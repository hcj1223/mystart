# 筛质数

### 1.朴素

```c++
int primes[N], cnt;
bool st[N];
void getprimes(int n)
{
    for (int i = 2; i <= n; i++)
        if (!st[i])
        {
            primes[cnt++] = i;
            for (int j = i + i; j <= n; j += i)
                st[j] = 1;
        }
}
```

### 2.线性

```c++
void get_primes(int n)
{
    for (int i = 2; i <= n; i++)
    {
        if (!st[i])
            primes[cnt++] = i;
        for (int j = 0; primes[j] <= n / i; j++)
        {
            st[primes[j] * i] = true;
            if (i % primes[j] == 0) //说明primes[j]一定是i的最小质因子
                break;
        }
    }
}
```
