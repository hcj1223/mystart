# 扩展欧几里得

#### *裴蜀定理：*

##### *给定任意一对正整数a, b,存在非零整数x, y,使得ax + by = (a, b)*

![扩展欧几里得](%E6%89%A9%E5%B1%95%E6%AC%A7%E5%87%A0%E9%87%8C%E5%BE%97.png)

```c++
int exgcd(int a, int b, int &x, int &y)
{
    if (!b) // a * x + 0 * y = gcd(a, b) = a
    {
        x = 1, y = 0;
        return a;
    }
    int d = exgcd(b, a % b, y, x); 
    y -= a / b * x;
    return d;
}
```
