# 容斥原理

```c++
int res = 0;
for (int i = 1; i < 1 << m; i++) // 状态压缩,第k位的数为 1 则表示选中第k个质数
{
    int t = 1, cnt = 0;
    for (int j = 0; j < m; j++)
    {
        if (i >> j & 1)
        {
            cnt++;
            if ((ll)t * p[j] > n) // 超出 1~n 的范围
            {
                t = -1;
                break;
            }
            t *= p[j];
        }
    }
    if (t != -1)
    {
        if (cnt & 1)
            res += n / t; // n/t 就是 1~n 中能被t整除数的个数
        else
            res -= n / t;
    }
}
```
