# 求所有约数

```c++
vector<int> getdivides(int n)
{
    vector<int> res;
    for (int i = 1; i <= n / i; i++)
        if (n % i == 0)
        {
            res.push_back(i);
            if (i != n / i)
                res.push_back(n / i);
        }
    sort(res.begin(), res.end());
    return res;
}
```
