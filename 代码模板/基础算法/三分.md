# 整数三分

```c++
while (r - l > 2)
{
    int mid1 = l + (r - l) / 3, mid2 = r - (r - l) / 3;
    if (check(mid1) < check(mid2)) //视情况而定
        l = mid1;
    else
        r = mid2;
}
```



# 实数三分

```c++
while (r - l > eps)
{
    double k = (r - l) / 3.0;
    double mid1 = l + k, mid2 = r - k;
    if (check(mid1) < check(mid2)) //视情况而定
        l = mid1;

}
```
